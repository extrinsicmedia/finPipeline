pymel.mayautils.getMayaAppDir¶

static mayautils.getMayaAppDir(versioned=False)¶

Determine the Maya application directory, first by checking MAYA_APP_DIR, then by trying OS-specific defaults.

if versioned is True, the current Maya version including ‘-x64’ suffix, if applicable, will be appended.
