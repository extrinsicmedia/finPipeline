pymel.core.modeling.polyMoveFacetUV¶

static modeling.polyMoveFacetUV(*args, **kwargs)¶

Modifies the map by moving all UV values associated with the selected face(s). The UV coordinates of the model are manipulated without changing the vertices of the 3D object.

Example:

import pymel.core as pm

# Opening up the textureView will make this example easier to visualize.

# Create a plane and select it.
pm.polyPlane( n='plg', sx=5, sy=5, w=10, h=10 )
pm.select( 'plg' )

# Display map borders.
pm.polyOptions( ao=True, dmb=True )

# Translate selected UVs as the vector (0.0, 1.0).
pm.polyMoveFacetUV( 'plgShape.map[30:35]', tu=0.0, tv=1.0 )

# Scale selected UVs as the scaling (0.5, 0.5);
pm.polyMoveFacetUV( 'plgShape.map[30:35]', su=0.5, sv=0.5 )
