pymel.core.modeling.polyMoveVertex¶

static modeling.polyMoveVertex(*args, **kwargs)¶

Modifies vertices of a polygonal object. Translate, rotate or scale vertices in local or world space.

Derived from mel command maya.cmds.polyMoveVertex

Example:

import pymel.core as pm

# Create a sphere.
pm.polySphere( r=5, n='plg' )

# Move vertices as the scaling vector (1.0, 1.5, 1.0).
pm.polyMoveVertex( 'plg.vtx[200:379]', 'plg.vtx[381]', s=(1.0, 1.5, 1.0) )

# Translate vertices as the vector (0.0, 2.0, 0.0).
pm.polyMoveVertex( 'plg.vtx[200:379]', 'plg.vtx[381]', ty=2.0 )
