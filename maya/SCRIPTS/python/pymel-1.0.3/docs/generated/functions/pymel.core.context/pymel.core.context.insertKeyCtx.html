pymel.core.context.insertKeyCtx¶

static context.insertKeyCtx(*args, **kwargs)¶

This command creates a context which may be used to insert keys within the graph editor

Flags:
Long name (short name) Argument Types Properties
breakdown (bd) bool
 
Specifies whether or not to create breakdown keys Flag can have multiple arguments, passed either as a tuple or a list.
exists (ex) bool
 
Returns true or false depending upon whether the specified object exists. Other flags are ignored.
history (ch) bool
 
If this is a tool command, turn the construction history on for the tool in question.
image1 (i1) unicode
   
image2 (i2) unicode
   
image3 (i3) unicode
 
Contexts support up to three icons that represent the tool associated with the context.
name (n) unicode
 
If this is a tool command, name the tool appropriately.

Derived from mel command maya.cmds.insertKeyCtx

Example:

import pymel.core as pm

# Create a insert key context for the graph editor
#
pm.insertKeyCtx( 'insertKeyContext' )
