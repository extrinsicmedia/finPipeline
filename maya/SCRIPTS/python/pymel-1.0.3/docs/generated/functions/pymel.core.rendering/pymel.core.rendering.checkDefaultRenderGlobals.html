pymel.core.rendering.checkDefaultRenderGlobals¶

static rendering.checkDefaultRenderGlobals(*args, **kwargs)¶

Command to check or set the changedmode for the defaultRenderGlobals node. In query mode, return type is based on queried flag.

Flags:
Long name (short name) Argument Types Properties
changed (ch) bool
 

Check or set the changedmode of the defaultRenderGlobals node. Flag can have multiple arguments, passed either as a tuple or a list.

Derived from mel command maya.cmds.checkDefaultRenderGlobals

Example:

import pymel.core as pm

# To check if the defaultRenderGlobals node is marked dirty or not.
#
pm.checkDefaultRenderGlobals( query=True, changed=True )
# Result: False #
# To mark the defaultRenderGlobals node dirty.
#
pm.checkDefaultRenderGlobals( changed=True )
# To mark the defaultRenderGlobals node not dirty.
#
pm.checkDefaultRenderGlobals( changed=False )
